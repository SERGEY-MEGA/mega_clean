## MegaCleaner

Два приложения, которые дополняют “Мониторинг системы”:

- **`MegaCleaner.app`**: большое окно в стиле мониторинга (процессы + метрики) + безопасные действия очистки.
- **`MegaCleanerMini.app`**: мини‑окно в менюбаре (процент свободного места + кнопки очистки) + кнопка запуска большого приложения.

### Что умеет (безопасно)

- **Кэш приложений**: очищает `~/Library/Caches` (не трогает `com.apple.*`).
- **Мусор**: удаляет старые логи (`~/Library/Logs` старше 14 дней) + чистит временную папку `NSTemporaryDirectory()`.
- **Корзина**: очищает `~/.Trash`.
- **ОЗУ**: без `sudo` (macOS сам управляет памятью; `purge` пробуется только если доступен без прав администратора).

### Сборка, запуск, установка в “Программы”

В папке проекта:

```bash
./build_and_run.sh
```

Скрипт:
- соберёт оба приложения в `build/`
- сгенерирует иконки (если их нет)
- попробует установить в `/Applications` (если нет прав — покажет команду с `sudo`)
- запустит мини‑окно
